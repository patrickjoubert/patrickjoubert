## What I'm building

**[Rippletide /dev](https://rippletide.com/dev)** — enforcement layer for Claude Code.

CLAUDE.md has no enforcement mechanism. Your agent knows your rules. It ignores them anyway.  
Rippletide intercepts every tool call before execution and blocks violations against a Context Graph built from your codebase. Not suggestions. Constraints.

→ `npx rippletide-code` · Open beta · Free · No API key · [Docs](https://docs.rippletide.com) · [Source](https://github.com/rippletideco/starter)
